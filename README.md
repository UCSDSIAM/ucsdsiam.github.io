# SIAM Student Chapter at UC San Diego Website 

Official source code for the SIAM Student Chapter at UC San Diego website, built with Vue 3 and Vite.

This project is similar in spirit to [the Competitive Programming Club at UC San Diego Website](https://github.com/icpcatucsd/icpcatucsd.github.io) and heavily reuses components from that repository.

---

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

---

## Codebase Overview

- **HTML entry point & metadata:** `index.html`
- **Root application layout:** `src/App.vue`
- **Routing:** `src/router/index.js`
- **Website data (content):** `src/data`
- **Page views:** `src/views`
- **Reusable components:** `src/components`
- **Global styles:** `src/assets/main.css`
- **Images & PDFs:** `public/assets`
